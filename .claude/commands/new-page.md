# New Page

Create a new Next.js App Router page at the given route.

## Instructions

1. Create `src/app/$ARGUMENTS/page.tsx`
2. Create `src/app/$ARGUMENTS/layout.tsx` if a layout is needed
3. Use TypeScript and server components by default
4. Add `"use client"` only if interactivity is required
5. Style with Tailwind CSS

## After writing the files

Run `npx tsc --noEmit` and fix any TypeScript errors before finishing.
Do not consider the task complete until the type check passes with zero errors.
